# Installation

## Требования
- Rust stable, cargo
- Рекомендуемые фичи: `mimalloc`, `metrics`, `tokio`

## Установка
```bash
cargo add nebula-memory
```

## Проверка окружения
```bash
rustc --version
cargo --version
```

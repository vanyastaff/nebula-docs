# Installation

## Требования
- Rust stable, `tokio`
- crates: `nebula-resource`, `serde`, `async-trait`, `thiserror`
- (опционально) `sqlx`, `reqwest`, `rdkafka`, `redis`

## Установка
```bash
cargo add nebula-resource async-trait serde serde_json thiserror
```

## Проверка окружения
```bash
rustc --version
cargo --version
```
